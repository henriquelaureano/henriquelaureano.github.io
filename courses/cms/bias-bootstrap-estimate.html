Bias bootstrap estimate

The definition of bias is given by:

\[ B(\hat{\theta}) = E(\hat{\theta}) - \theta. \]

Thus, the bias bootsrap estimate is:

\[ \hat{B(\hat{\theta})} = \bar{\hat{\theta}^{*}} - \theta, \]

expression in which \(\bar{\hat{\theta}^{*}} = \frac{1}{B} \sum_{b = 1}^{B} \hat{\theta}^{(b)}\) and \(\hat{\theta}\) is the estimate calculated using the original sample.
